Version 2.1.9

- CHG: Added EventData 'jobNumber' column to optional CustomFieldInfo field group. 
- CHG: Added Device 'driverID' column. 
- FIX: Fixed 'getKeyCode' function in 'utils.js' (thanks to Pierluigi Bucolo).
